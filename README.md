
The application for a recruiter specializing in hiring programmers allows companies to register accounts and define requirements for desired candidates. Programmers can create an account, complete their CV, and suspend job searches. Company representatives can browse candidate profiles, mark them as hired, and manage company accounts. Ensuring user-friendly application use and data security is crucial.


The technologies used during the development of the application include: Spring Framework (including Spring Boot, Spring Data JPA, Spring Web, Spring Security, Spring Webflux), Thymeleaf for server-side HTML rendering and templating, Flyway for database schema management and migrations, MapStruct for object mapping in Java, and Springdoc OpenAPI for generating OpenAPI documentation and facilitating API interface creation and testing
